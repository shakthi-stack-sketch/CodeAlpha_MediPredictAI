About MediPredict AI

Learn how the prediction system works behind the scenes.

Dataset

This application uses the UCI Heart Disease (Cleveland) dataset, a widely used public dataset containing patient medical records and heart disease diagnoses.

  • Records: {{ metadata.records | default('297') }} patient samples
  • Features: 13 clinical measurements
  • Target: Binary heart disease presence (Yes/No)
  • Source: UCI Machine Learning Repository

Machine Learning Model

A {{ metadata.model_name | default('Random Forest') }} classifier is trained on the heart disease dataset to predict disease risk from patient features.

  • Algorithm: {{ metadata.model_name | default('Random Forest') }}
  • Accuracy: {{ (metadata.accuracy * 100) | round(1) if metadata.accuracy is number else metadata.accuracy | default('N/A') }}%
  • Preprocessing: StandardScaler normalization
  • Validation: 80/20 stratified train-test split

Workflow

1

Data Collection

Heart disease patient records are loaded from the UCI Cleveland dataset and cleaned.

2

Model Training

Random Forest and Logistic Regression models are trained; the best performer is selected and saved.

3

User Input

Users enter clinical measurements through the prediction form on the web interface.

4

Prediction

Input data is scaled and passed to the model, returning risk level, confidence, and recommendations.
